**第一阶段：Emacs 基础入门**

**学习目标**  
掌握 Emacs 的安装、启动与基本界面操作；能够打开、编辑、保存文件和退出程序；理解缓冲区和窗口的基本概念。

**具体任务节点**  
1. 安装 Emacs（根据你的操作系统选择对应版本）。  
2. 启动 Emacs，观察启动界面和菜单栏。  
3. 学习如何新建、打开、保存文件（`C-x C-f`，`C-x C-s`，`C-x C-w`）。  
4. 学习移动光标的常用快捷键：上下左右、行首行尾、单词跳跃。  
5. 学习撤销与重做（`C-x u`，`C-/` 或 `C-_`）。  
6. 学习复制、剪切、粘贴（`C-w`，`M-w`，`C-y`）。  
7. 学习退出 Emacs（`C-x C-c`）。  
8. 理解缓冲区（buffer）与窗口（window）的区别，并尝试切换缓冲区（`C-x b`）。

**推荐查阅知识点关键词**  
Emacs 安装、基本快捷键、文件操作、缓冲区、退出

**实践任务**  
- 创建一个文本文件，输入至少三行内容，保存后关闭，再重新打开确认内容。  
- 在编辑过程中故意犯错，使用撤销恢复。

**验收标准**  
- 能成功启动 Emacs 并打开任意文件。  
- 能使用快捷键完成基本的文件创建、编辑、保存、退出。  
- 能解释什么是 Emacs 的缓冲区，并列出至少3个基本移动快捷键。

**第二阶段：文本操作与基本配置**

**学习目标**  
掌握 Emacs 的搜索替换、多文件编辑、窗口管理；了解 .emacs 配置文件的结构，学会设置基本选项（如缩进、行号）。

**具体任务节点**  
1. 学习增量搜索（`C-s` 向前，`C-r` 向后）与替换（`M-%`）。  
2. 学习分屏操作：水平分屏（`C-x 2`）、垂直分屏（`C-x 3`），切换窗口（`C-x o`），关闭窗口（`C-x 0`）。  
3. 学习在同一个 Emacs 中编辑多个文件（通过 `C-x C-f` 打开不同文件，使用 `C-x b` 切换缓冲区）。  
4. 了解 `.emacs` 或 `init.el` 配置文件的位置（Linux/OS X：`~/.emacs` 或 `~/.emacs.d/init.el`；Windows：`~/.emacs.d/init.el`）。  
5. 在配置文件中添加一行：`(setq linum-mode t)`（Emacs 25 以下）或 `(global-linum-mode t)`（旧版），新版 Emacs（26+）使用 `(global-display-line-numbers-mode t)`。  
6. 学习如何设置默认缩进为 4 个空格：`(setq-default indent-tabs-mode nil)` 和 `(setq-default tab-width 4)`。  
7. 学会重新加载配置文件：`M-x eval-buffer` 或重启 Emacs。

**推荐查阅知识点关键词**  
搜索替换、窗口管理、缓冲区、点文件（dotfile）、Emacs Lisp 基础、缩进设置

**实践任务**  
- 打开至少两个文件，使用分屏同时查看，并在一个窗口中搜索另一个窗口中的内容。  
- 修改你的配置文件，启用行号显示并将缩进设为 4 个空格。  
- 使用增量搜索在一个中型文本文件中查找特定单词，并执行一次替换（注意确认替换）。

**验收标准**  
- 能熟练使用 `C-s`/`C-r` 进行实时搜索，使用 `M-%` 进行替换。  
- 能创建/关闭分屏，并在多个窗口间切换。  
- 配置文件内容正确，重启 Emacs 后行号显示且缩进为 4 个空格。  

完成这些任务后，输入 **继续**。
