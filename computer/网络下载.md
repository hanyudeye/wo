如何下载网络 ts 视频片段并合并为整个视频


要下载网络 TS 视频片段并合并为完整视频，通常步骤如下：

1. **获取 m3u8 地址**：打开浏览器开发者工具（F12），在“网络”标签页过滤 `m3u8`，找到主播放列表地址。  
2. **下载所有 TS 片段**：使用 `ffmpeg`、`yt-dlp` 或 `m3u8dl` 等工具。  
   - 例如用 `yt-dlp`（推荐）：  
     ```bash
     yt-dlp -f best "https://example.com/playlist.m3u8" -o output.mp4
     ```  
   - 或用 `ffmpeg`：  
     ```bash
     ffmpeg -i "https://example.com/playlist.m3u8" -c copy output.mp4
     ```  
3. **若已有本地 .ts 文件**，直接用 `ffmpeg` 合并：  
   ```bash
   ffmpeg -i "concat:file1.ts|file2.ts|..." -c copy output.mp4
   ```  
   或在 Windows 下用 `copy /b *.ts output.ts`。

注意：某些网站可能加密 TS 片段，需要先解密（通常需要 AES-128 key），此时 `yt-dlp` 或专用工具会自动处理。
