# 如果从 **2026 年往后 5～10 年** 看，我不建议普通人学那些：

* 门槛低
* 容易外包
* 容易被 AI 替代

的技能。

应该优先学：

```text
AI难以完全替代
市场长期有需求
可以独立赚钱
能与AI协作放大产出
```

结合你的情况：

* 计算机专业
* 对 AI 感兴趣
* 会折腾工具
* 想做副业
* 愿意长期学习

我会这样排序。

---

# 第一梯队（最推荐）

## AI自动化开发

例如：

* n8n
* Python
* API调用
* AI Agent
* 工作流自动化

你不用成为顶级程序员。

只要能帮企业实现：

```text
自动发内容
自动处理表格
自动客服
自动数据收集
```

就有价值。

未来很多小企业会需要这种人。

```text
利用现有模型解决问题
```

* AI知识库
* AI客服
* AI内容工具
* AI办公工具
