# 年夜饭

以下是一份适合6-8人食用的江浙家常菜年夜饭菜单：

### 凉菜
- **金陵盐水鸭**：选用当年仔鸭，用花椒盐腌制后入清卤浸泡，再冷水下锅焖煮，捞出晾凉切块。其肉质紧实，咸香适口。
- **凉拌海蜇头**：海蜇头泡发后切丝，加入黄瓜丝、香菜、蒜末、香醋、糖拌匀，口感脆爽，寓意年年有余。

### 热菜
- **苏式松鼠鳜鱼**：鳜鱼去骨改菱形花刀，腌制后拍淀粉炸至金黄，再浇上用番茄酱、糖、醋等熬制的酱汁，造型美观，味道酸甜可口。
- **无锡酱排骨**：猪肋排切寸段焯水后，煸炒至微黄，加入绍酒、酱油、冰糖等煨煮至酱汁浓稠挂骨，浓油赤酱，甜咸适中。
- **杭三鲜**：准备水发肉皮、鱼丸、猪肉、熟鸡肉、河虾、冬笋等食材。将冬笋煮熟切片，猪肉剁碎搓成小丸子蒸熟，再将各食材一起放入锅中煸炒，加入调料和上汤烧滚，调入蔬之鲜放入鱼丸煮开，勾芡后撒上葱花即可。
- **红烧狮子头**：猪肉剁成肉泥，加入藕末、鸡蛋、葱姜水等搅拌上劲，搓成肉丸炸至金黄后红烧，肉质紧实有弹性，味道鲜美。
- **蒜蓉西兰花**：西兰花洗净切成小朵，焯水后捞出备用。锅中热油，放入蒜蓉爆香，加入西兰花翻炒，加盐、少许鸡精调味即可，清爽可口，寓意清雅高洁。
- **香菇扒菜心**：菜心对半切开，与泡发的香菇分别焯水。用蒜末爆香，加入蚝油、生抽等调料，放入香菇小火煨煮，再放入菜心同烧，勾薄芡，寓意花开富贵。

### 汤品
- **鲜菌豆腐丸子汤**：肉末加豆腐、鸡蛋等搅拌上劲，锅中清水加姜片和各类鲜菌煮开，转小火将肉馅挤成丸子下锅，煮至丸子全部浮起后调味，撒上葱花，寓意阖家团圆。

### 主食
- **八宝饭**：可直接购买成品，上锅蒸20分钟，倒扣在盘中。也可自制，碗底用红枣、莲子等摆出花样，填入拌了猪油和白糖的糯米饭，压实蒸熟后倒扣，寓意生活甜蜜。

## 二、分步做法（按“先备菜、后烹饪”顺序，省时高效）
### 【提前备菜】（年夜饭前1天/当天上午做，减少晚上忙碌）
1. **盐水鸭处理（买生鸭版）**
   - 粗盐+花椒下锅小火炒出香味，放凉后均匀抹在鸭身内外，腌制6小时（放冰箱）。
   - 冷水下锅，加姜片、葱段、料酒，大火烧开后转小火焖煮40分钟，关火焖20分钟。
   - 捞出晾凉，切块装盘（买成品直接切）。
2. **海蜇头泡发**
   - 干海蜇头用清水泡6小时，期间换3次水，泡到无咸味后切丝；黄瓜切丝，大蒜切末。
3. **八宝饭准备（自制版）**
   - 糯米泡4小时，沥干后上锅蒸30分钟，趁热加猪油+白糖拌匀。
   - 碗底抹一层油，摆上红枣、莲子、核桃、葡萄干，铺一层糯米饭，压平后铺红豆沙，再盖一层糯米饭压实。
   - 上锅蒸20分钟，倒扣在盘中即可（成品直接蒸15分钟）。
4. **狮子头肉馅调制**
   - 猪肉馅+莲藕末+鸡蛋+葱姜水+盐3g+生抽10ml，朝一个方向搅拌上劲，搓成4个大丸子（约120g/个）。

### 【年夜饭当天烹饪】（按“凉菜→热菜→汤品”顺序）
#### ▶ 凉菜（开饭前30分钟做）
- **凉拌海蜇头**
  海蜇丝+黄瓜丝+香菜段+蒜末，加香醋30ml+白糖10g+盐2g+香油5ml，拌匀装盘，脆爽解腻。

#### ▶ 热菜（按“费时间→快手”顺序做）
1. **无锡酱排骨（最先做，炖的时候做其他菜）**
   - 肋排焯水去血沫，捞出沥干；锅中放少许油，下冰糖炒出焦糖色。
   - 倒入肋排翻炒上色，加姜片+料酒+生抽+老抽，再加开水没过排骨。
   - 大火烧开转小火炖1小时，最后大火收汁，直到酱汁浓稠挂在排骨上。
2. **红烧狮子头**
   - 锅中倒油，油温六成热，放入搓好的狮子头，炸至表面金黄捞出。
   - 锅中留底油，加姜片爆香，放狮子头，加生抽+老抽+开水没过狮子头，小火炖40分钟，收汁即可。
3. **苏式松鼠鳜鱼（颜值担当，最后做热菜）**
   - 鳜鱼去鳞去内脏，切去鱼头，鱼身去骨，在鱼肉上切菱形花刀（别切断鱼皮）。
   - 加料酒+盐2g腌制10分钟，均匀拍上玉米淀粉，抖掉多余粉末。
   - 油温七成热，先炸鱼头定型，再炸鱼身，炸至金黄酥脆捞出，摆成松鼠造型。
   - 锅中留底油，放番茄酱+白糖+香醋+少许清水，熬成酸甜酱汁，淋在鱼身上，撒葱花。
4. **杭三鲜**
   - 鸡胸肉切片焯水；河虾剪去虾须焯水；冬笋切片焯水去涩。
   - 猪肉馅加鸡蛋+盐1g，搓小丸子蒸熟；水发肉皮切块。
   - 锅中倒油，放姜片爆香，加所有食材翻炒，倒入高汤（或清水）烧开。
   - 加盐3g调味，勾薄芡，撒葱花出锅。
5. **蒜蓉西兰花（快手菜，最后炒）**
   - 西兰花小朵焯水1分钟（水中加少许盐+油，保持翠绿），捞出沥干。
   - 锅中倒油，蒜末爆香，倒入西兰花翻炒，加盐2g调味，出锅。
6. **香菇扒菜心**
   - 菜心焯水捞出摆盘；泡发的香菇挤干水分。
   - 锅中倒油，蒜末爆香，加香菇+泡香菇的水+蚝油+生抽，小火煨5分钟。
   - 勾薄芡，将香菇和酱汁浇在菜心上。

#### ▶ 汤品（热菜快做好时煮）
- **鲜菌豆腐丸子汤**
  - 嫩豆腐碾碎，加猪肉馅+鸡蛋+盐2g+葱花，搅拌上劲，搓小丸子。
  - 锅中加清水+姜片烧开，放入金针菇+蟹味菇煮5分钟。
  - 转小火，将小丸子逐个下入锅中，煮至丸子浮起。
  - 加盐3g调味，撒葱花出锅。

## 三、年夜饭省时小贴士
1. **半成品替代**：松鼠鳜鱼的鱼丸、狮子头的肉馅，可直接买超市半成品，节省时间。
2. **烹饪顺序**：先炖酱排骨、狮子头（费时间），中间穿插做杭三鲜，最后炒快手青菜、煮鱼，避免菜凉。
3. **解腻搭配**：浓油赤酱的菜多，搭配凉拌海蜇头、西兰花，清爽不腻。


# 欧美饮食

如果参考欧美近几年比较主流的健康饮食体系，例如：

- 高蛋白
- 高纤维
- 低糖
- 少加工食品
- 大量蔬菜
- 适量水果
- 优质脂肪（橄榄油、坚果、鱼类）

而且做法其实比中餐简单很多。
