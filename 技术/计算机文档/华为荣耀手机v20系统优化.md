## 打开开发者模式

设置 → 关于手机 → 连续点版本号7次-> 开发者选项 → USB调试

允许 USB 调试？ RSA 指纹 
点 始终允许

## 安装 adb

scoop install adb

连接测试：

adb devices

正常显示：
PCTXXXX device usb:1-1 product:kirin980 model:V20

## 进行清理

Step1 查看全部：
adb shell pm list packages > packages.txt
> 导出所有的包


Step2 安全删除：
adb shell pm uninstall --user 0 包名

Step3 禁用风险组件： (先禁用，稳定后执行上面的删除)
adb shell pm disable-user --user 0 包名

Step4 优化动画：
adb shell settings put global window_animation_scale 0.5
adb shell settings put global transition_animation_scale 0.5
adb shell settings put global animator_duration_scale 0.5

流畅度明显提升。

极致优化（高手才做）

关闭日志：

adb shell setprop log.tag.HwLog VERBOSE
adb shell setprop log.tag.HwModuleLog VERBOSE

关闭统计：

adb shell pm disable-user --user 0 com.huawei.bd

关闭智能推荐：

adb shell pm disable-user --user 0 com.huawei.recsys
